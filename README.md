# genshin-mys-checkin

原神米游每日社签到

- 每天早上 7:00 ~ 7:10 随机时间签到
- 支持多帐号及多角色
- 有角色签到失败不会使整体流程终止，并且在执行完毕后你会收到一封来自 GitHub 的 Actions 失败提醒邮件

## Usage

懂的都懂

## Secrets

- `COOKIE`  
  米游社 cookie，如需多帐号签到，以 `#` 隔开
